# 🌐 Developer Portfolio Website

A modern, fully responsive **developer portfolio website** built with **Next.js**, **Tailwind CSS**, and **Node.js**.  
This project showcases my professional experience, technical skills, and featured projects with a clean UI and smooth animations.

The website includes a **functional contact form powered by Node.js and Nodemailer** for sending emails directly from the site.

---

## 🚀 Tech Stack

Frontend:
- Next.js
- React.js
- Tailwind CSS
- JavaScript / TypeScript
- Framer Motion (for animations)

Backend:
- Node.js
- Express.js
- Nodemailer

Tools:
- Git & GitHub
- Vercel (Deployment)
- Responsive Design

- Modern UI/UX design
- Fully responsive across devices
